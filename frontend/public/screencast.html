Скринкаст без Xcode

Приложение внутри рамки iPhone с Dynamic Island. Запись — обычным захватом окна в QuickTime или OBS, без симулятора.

  1. Запустите npm run dev и один раз войдите в приложение (если ещё не вошли).
  2. Нажмите «Окно для записи» — откроется чистое окно только с телефоном.
  3. QuickTime → «Файл» → «Новая запись экрана» → выберите это окно. Или OBS → «Захват окна».
  4. Кликайте внутри экрана телефона — это живое приложение.

Альтернатива: запись экрана на реальном iPhone (Control Center) и AirDrop на Mac — тоже с вырезом под остров.
