# 3.5 Социальные битвы бюджетов

Групповые челленджи с друзьями.

## Типы челленджей

- `least_spend` — кто меньше потратил
- `most_saved` — кто больше отложил
- `streak` — самая длинная серия без нарушений

## Технически

- Redis sorted sets для реал-тайм лидербордов
- Совместные цели через goal-service
