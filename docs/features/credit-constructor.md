# Ипотечный разбор (Кредитный конструктор)

> Продукт: [monetization.md](../product/monetization.md) — основная монетизация.

## Цель

Единый ответ на «потяну ли ипотеку» + сравнение банков и сценарии «сейчас / подождать».

## Выход системы

1. Шанс одобрения (высокий / средний / низкий) + причины
2. Безопасный предел (сумма, комфортный платёж)
3. Риск нагрузки (подушка после платежа, хрупкость)
4. Сценарий «если взять сейчас»
5. Сценарий «если подождать»
6. Сравнение 3–5 банков
7. Оптимальный вариант

## Реализация

| Компонент | `back` |
|-----------|--------|
| DTI, кредиты | `credit-service` |
| Ставки, банки | `bank-service` |
| Симуляции | `analytics-service` — `/api/v1/scenarios/` |
| UI | `front` — `/credits`, TimeMachine chart |
