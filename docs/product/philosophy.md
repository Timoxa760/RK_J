# Философия продукта

## Принципы

- **Мягкая** подача — без оценки личности и давления
- **Ненавязчивая** — одно действие, не десять задач
- **Ответы, не графики** — выводы и последствия вместо таблиц

## Формулировки

| Избегать | Использовать |
|----------|--------------|
| Вы плохо тратите | Можешь улучшить |
| Критическая ошибка | Вот как складывается динамика |
| Вы должны | Если ничего не менять, путь будет таким |
| DTI превышен | Ипотека сейчас сделает ситуацию более хрупкой |

## Визуальный язык (`front`)

- Спокойная палитра (stone, flow-линии на welcome)
- ECharts — поддержка narrative, не «бухгалтерский отчёт»
- Hero: «Поток» — голосовой / навигаторный месседж
