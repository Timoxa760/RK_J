# Roadmap и трудозатраты (post-MVP)

> Оценки для защиты хакатона. 1 dev-week ≈ 40 ч.

## Q3 2026 — MVP → продукт

| Эпик | Содержание | Оценка |
|------|------------|--------|
| Онбординг UI | `/onboarding`, PATCH profile, complete flag | 3–5 д |
| Narrative dashboard | тексты вместо голых графиков | 3–5 д |
| API sync | DTI %, timemachine `points`, credits dashboard | 2–3 д |
| Голос multi-tx | одна фраза → N транзакций (LLM) | 5–8 д |
| Ипотечный разбор UI | bank compare, now/wait | 5–7 д |

**Итого Q3:** ~18–28 dev-days

## Q4 2026 — рост

| Эпик | Оценка |
|------|--------|
| Push / digest email | 3 д |
| Мобильная PWA polish | 3 д |
| A/B формулировок рекомендаций | 5 д |

## 2027 — масштаб

| Эпик | Оценка |
|------|--------|
| Open Banking (опционально, вне scope кейса) | 20+ д |
| B2B white-label для банков/МФО | 30+ д |
| Платный ипотечный модуль (monetization) | 10 д |

## Монетизация

Бесплатно: фин. здоровье, цель, базовые инсайты.  
Платно: [ипотечный разбор](./monetization.md) — сравнение банков, сценарии now/wait.

## Ценность

| Для пользователя | Для бизнеса |
|------------------|-------------|
| Ответы без бухгалтерии | Freemium → платные сценарии |
| Решения по кредиту/ипотеке | Партнёрства с банками (CPA) |
| Привычка 1 действие/неделю | Retention выше классических трекеров |
