# Финансовое здоровье

## Что это

Показатель состояния финансов пользователя: устойчивость, запас денег, движение к цели.

Бесплатный базовый слой — основа для всех прогнозов и платных сценариев (ипотека, крупные покупки).

## Из чего состоит

| Блок | Поля |
|------|------|
| Доходы | активный, пассивный |
| Расходы | фактические (голос, чеки, ФНС), обязательные (если указаны) |
| Подушка | сумма, ликвидность (опционально) |
| Цель | сумма, прогресс, срок |

## Основные метрики

| Метрика | Описание |
|---------|----------|
| **Финансовая устойчивость** | Шкала 0–100 или 🟢/🟡/🔴 |
| **Runway** | Месяцев можно жить без дохода при текущих расходах |
| **Свободный денежный поток** | Доход минус расходы |
| **Скорость цели** | Время до достижения при текущем поведении |
| **Индекс привычек** | Импульс, регулярность обязательств |

## Как показывается (без давления)

- «У тебя есть запас на X месяцев»
- «Ты движешься к цели»
- «Текущие траты замедляют / ускоряют путь»
- «Структура доходов стабильная / нестабильная»

Формулировки: «можешь улучшить», «вот как складывается динамика», «если ничего не менять, путь будет таким».

## Реализация

| Слой | `back` | `front` |
|------|--------|---------|
| DTI, подушка, кредиты | `credit-service` — `/api/v1/credits/` | `/credits` |
| Dashboard, sankey | `receipt-service` — `/api/v1/dashboard/*` | `/dashboard` |
| Инсайты | `analytics-service` — `/api/v1/insights/` | `/analytics`, InsightsPanel |
