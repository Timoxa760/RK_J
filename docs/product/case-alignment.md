## Три направления из ТЗ

| Направление Клерка | «Поток» | Статус |
|--------------------|---------|--------|
| **1. Кредитный светофор** | DTI, стресс-тест, «потяну ли кредит» | ✅ `/credits` |
| **2. Финансовый диагноз** | Онбординг ~1 мин + одно действие | 🟡 docs ✅, `/onboarding` ⏳ |
| **3. Трекер накоплений** | Одна цель, прогноз, Time Machine | 🟡 API + charts |

Ограничение «без интеграции с банками» — **соблюдаем**: нет Open Banking; `bank-service` — сравнение условий ипотеки, не личный счёт.

## Чего не делаем (ТЗ)

| Запрет | Как соблюдаем |
|--------|---------------|
| Бухгалтерский UI | Narrative, не CRUD таблиц |
| CRUD без ценности | Каждый экран → ответ или действие |
| AI ради AI | Голос/парсер → модель → влияние на цель |

## Demo для жюри (6 актов)

1. Welcome — «ответы, не графики»
2. `/onboarding` — доход + цель
3. Голос: «продукты 5000, кроссовки 16000»
4. Dashboard — один текстовый инсайт
5. `/credits` — кредитный светофор
6. «Потяну ли ипотеку?» — [monetization.md](./monetization.md)
